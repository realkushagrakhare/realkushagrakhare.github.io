Real Engine 2.0

A 3D Game Engine in progress using LWJGL and new OpenGL 3.0+ using the programmable pipeline.

    Some of the features developed
  • VAOs, VBOs, Vertex and Fragment Shaders for optimization
  • OBJ and PLY file reader to load 3D models
  • Multiple point lights and upto 4 directional lights as well as Specular Lighting
  • Terrain generation via heightmaps as well as procedural terrain using Perlin noise
  • Post Processing effects like lensflare, fog, cel shading, gaussian blur, contrast changer etc.
  • Rendering GUIs and distance field text rendering
  • Particle effects like bloom, fire with particle life cycle
  • Soft shadows using shadow mapping
  • Specular maps, bump maps and glow maps to specifiy object's shininess, normals and glow respectively
  • Created a OBJ and PLY file reader
  • Mouse picker to place objects in the scene
  • Textures with texture atlas and transparency
  • Skybox with Day-Night cycle
